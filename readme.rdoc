=== Start dev server:
  npm run build-css
  npm run dev
Note for debug: "npm run dev debug"

=== Start prod server:
  npm run build
  npm start

=== Run tests:
  npm test

=== Database commands
  npm run db:reset
  npm run db:migrate
  npm run db:seed
  npm run db:drop
